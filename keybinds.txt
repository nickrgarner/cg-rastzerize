Keybinds

Change View

a - translate view left along view X		key 65
d - translate view right along view X		key 68
w - translate view forward along view Z		key 87
s - translate view backward along view Z	key 83
q - translate view up along view Y		key 81
e - translate view down along view Y		key 69
A - rotate view left around view Y (yaw)	shift = key 16	
D - rotate view right around view Y (yaw)
W - rotate view forward around view X (pitch)
S - rotate view backward around view X (pitch)


Select Model

left - select and highlight previous triangle set	key 37
right - select and highlight next triangle set		key 39
up - select and highlight next ellipsoid		key 38
down - select and highlight previous ellipsoid		key 40
spacebar - deselect and turn off highlight		key 32


Transform Model

k - translate selection left along view X		key 75
; - translate selection right along view X		key 186		
o - translate selection forward along view Z		key 79
l - translate selection backward along view Z		key 76
i - translate selection up along view Y			key 73
p - translate selection down along view Y		key 80
K - rotate selection left around view Y (yaw)		
: - rotate selection right around view Y (yaw)
O - rotate selection forward around view X (pitch)
L - rotate selection backward around view X (pitch)
I - rotate selection clockwise around view Z (roll)
P - rotate selection counterclockwise around view Z (roll)
